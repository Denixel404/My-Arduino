# My-Arduino
## Сборник моих разных скетчей для платы Arduino

В этом репозитории содержатся все мои скетчи, созданные 
при работе с программируемой платой Arduino Nano как в учебных, 
так и в проектных целях.

## Описание каждого файла
Ниже перечислены все файлы репозитория с кодом, их предназначение и описание работы.
Каждый файл представляет отдельную, самостоятельную программу, загружаемую на плату Arduino.

### Smooth_RGB.ino
Код для РГБ светодиода, который при подключении питания будет плавно менять цвет.

### chip_74CH596.ino
Пример кода для работы с микросхемой 74CH596, который кодирует десятичные числа 
в двоичные и отображает их в виде восьми горящих светодиодов.

### run_lights.ino
Бегущие огоньки.

### temperature_display.ino
Код для простой метеостанции, считавающей показания с датчика температуры и влажности 
воздуха, и выводящей их на дисплей. В примере используется датчик DHT11 и символьный 
дисплей LCD1602 с модулем I2C.

### traffic_lights.ino
Код, имитирующий работу светофора для людей и машин. Рекомендую использовать светодиоды 
красного, жёлтого и зелёного цветов.
